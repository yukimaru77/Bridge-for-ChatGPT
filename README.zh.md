# Bridge for ChatGPT (简体中文)

**Bridge for ChatGPT** 是一款能让你在使用母语的同时，享受英语提示词带来的“高智商”回答的扩展插件。

## 🚀 简介
你知道吗？如果用英语提问，ChatGPT 通常能提供逻辑更清晰、精度更高的回答。
本插件会自动将你的母语输入翻译成英语发送给 ChatGPT，以最大限度地激发 AI 的潜能。当收到英语回复时，你只需点击“翻译”按钮，即可将其转换回你的母语（中文等）。

## ✨ 主要功能
* **提示词自动翻译:** 支持中文直接输入，插件会自动将其转换为英语并发送。
* **回复一键翻译:** ChatGPT 的回复旁会出现“翻译（Translate）”按钮，点击即可将英语回复译回中文。
* **保留原始格式:** 翻译后依然完美保留 Markdown 格式、代码高亮以及 KaTeX 数学公式。
* **便捷开关:** 聊天界面设有开关，可随时启用或禁用翻译功能。
* **基于 Gemini API:** 利用 Google Gemini 模型实现快速、自然的翻译体验。

## 🛠 前置要求
使用此扩展需要免费的 **Google Gemini API Key**。
* [点击此处获取 API Key (Google AI Studio)](https://aistudio.google.com/app/apikey)

## ⚙️ 安装与设置

1.  **安装扩展**
    * 从 Chrome 网上应用店下载安装。

2.  **配置设置 (Options)**
    * 点击浏览器工具栏的拼图图标 🧩，打开 Bridge for ChatGPT 的“选项”。
    * **Your Main Language**: 选择你希望阅读的语言（例如：Chinese Simplified / 简体中文）。
    * **Gemini API Key**: 粘贴你申请到的 API Key。
    * 点击 **Save Settings** 保存。

3.  **使用方法**
    * 打开 [chatgpt.com](https://chatgpt.com)。
    * 确输入框附近的“Bridge”开关处于 **ON** 状态。
    * 直接使用中文输入并发送即可。

## 🔒 隐私政策
* **数据处理:** 用户的输入文本仅为了翻译目的发送至 Gemini API。
* **无数据存储:** 开发者不会在服务器上保存你的提示词、对话记录或任何个人信息。
* **直接通信:** 所有数据传输均在你的浏览器与 API 提供商（OpenAI/Google）之间直接进行。
